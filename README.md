## 🗂️ Topic Breakdown

### 🔢 Array
- **Sorting**: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort (Ascending & Descending)
- **Searching**: Find duplicates, missing numbers, unique elements, first & last occurrence
- **Subarrays**: Kadane's Algorithm (max subarray sum), count subarrays with sum K, longest positive subarray
- **Two Pointers**: Two Sum, 3 Sum, 4 Sum (brute, better, optimal)
- **Classics**: Dutch National Flag, Moore's Voting (majority element), Pascal's Triangle, spiral matrix, rotate matrix
- **Hashing**: Character hashing, integer hashing, hash arrays

### 🔍 Binary Search
- Lower Bound, Upper Bound, Search Insert Position
- Search in Rotated Sorted Array (with/without duplicates)
- Find Minimum in Rotated Sorted Array, Number of Rotations
- Find Peak Element, Mountain Array
- Find Sqrt, Nth Root
- Koko Eating Bananas, Aggressive Cows

### 🧮 Bit Manipulation
- Binary ↔ Decimal conversion
- Check, Set, Clear, Toggle i-th bit
- Remove Lowest Set Bit

### 🌐 Graph
- **Representations**: Adjacency Matrix & Adjacency List (directed/undirected, weighted/unweighted)
- **Traversals**: BFS, DFS (recursive)
- **Applications**: Cycle detection (BFS & DFS), Number of Islands, Number of Provinces, Rotten Oranges

### 🔗 Linked List
- **Singly Linked List**: Create, insert (head/tail/position), delete (head/tail/any/duplicate), reverse, traverse
- **Doubly Linked List**: Create, insert, delete, reverse
- **Problems**: Add Two Numbers, Odd-Even LL, Sort 0-1-2, Y-intersection point, Hare & Tortoise (cycle detection)

### 🔢 Number Theory
- GCD (Euclidean), LCM
- Sieve of Eratosthenes, Smallest/Largest Prime Factor
- Prime Factorization (brute & sieve-based)
- Fast/Modular Exponentiation, Big Mod
- Euler's Totient Function
- Divisor counting & summation

### 📦 Stack & Queues
- **Implementations**: Stack (Array, Linked List), Queue (Array, Linked List, Circular)
- **Expression Conversion**: Infix → Postfix, Infix → Prefix, Prefix → Infix
- **Problems**: Balanced Parentheses, Next Greater Element, Next Smaller Element, Insert at Bottom

### 🌳 Tree
- **Binary Tree**: Implementation, recursive & iterative traversals (Preorder, Inorder, Postorder), Level Order, height, size, count leaves
- **BST**: Recursive implementation, search
- **AVL Tree**: Implementation with rotations
- **Heap**: Max Heap, Heapify, Heap Sort

### 🏢 Object Oriented Programming
- Classes, objects, constructors, `this` keyword
- Access modifiers, encapsulation
- Single & multiple inheritance
- Function overloading
- Dynamic memory allocation & object pointers
- Operator overloading (fraction addition)

## ⚙️ How to Run

### Prerequisites
- A C++ compiler (GCC / MSVC / Clang)
- C++17 or later recommended

### Compile & Run
```bash
# Using g++
g++ -std=c++17 -o output filename.cpp
./output

# Using MSVC (Developer Command Prompt)
cl filename.cpp
filename.exe
```

### With VS Code
Use the preconfigured build task:
```
Ctrl + Shift + B → build
```
